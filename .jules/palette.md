## 2024-05-24 - Dynamic ARIA labels on state-toggling icon-only buttons\n**Learning:** Icon-only buttons that toggle state (like show/hide password) need dynamic `aria-label` attributes that reflect the current state and action, as static labels or missing labels leave screen reader users unaware of the button's purpose or current state.\n**Action:** Always implement dynamic `aria-label` attributes reflecting current state (e.g., `aria-label={showPassword ? 'Hide password' : 'Show password'}`) for state-toggling icon-only buttons to ensure full accessibility.
