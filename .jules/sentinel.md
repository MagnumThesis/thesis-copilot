## 2026-03-12 - Prevent IDOR on Chat Messages Endpoint\n**Vulnerability:** Insecure Direct Object Reference (IDOR) on `/api/chats/:id/messages`\n**Learning:** The `getMessagesHandler` in `src/worker/handlers/messages.ts` fetched chat messages solely based on the `id` parameter without verifying if the authenticated user actually owned the chat.\n**Prevention:** Always extract the user's identity from the authentication token (`getUserIdFromToken`) and explicitly include an ownership check (`.eq('user_id', userId)`) when querying resource items that belong to specific users.
